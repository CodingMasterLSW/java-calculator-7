# 문자열 덧셈 계산기

## 기능 구현 목록

---

### 기본 구분자 처리
- [x] `,` 혹은 `:`를 구분자로 구분하기 


### 커스텀 구분자 처리
- [x] 커스텀 구분자를 사용해 구분하기
  - [x] 커스텀 구분자가 한 자리일 경우
  - [x] 커스텀 구분자가 여러 자리일 경우 
  
    
### 예외 처리
- [x] 양수가 아닌 값이 입력될 경우 예외 발생
  -[x] 0일 경우 예외 발생
  -[x] 음수일 경우 예외 발생
  -[x] 문자열이 입력되었을 때 예외 발생
  -[ ] int값을 넘어갔을 경우에는, 예외가 발생해서는 안 된다.


### 계산 기능
-[x] 추출한 값을 더하기
-[x] 더한 값을 출력하기
